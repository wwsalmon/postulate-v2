# Postulate v2

Rebuild of Postulate
- Username/password sign in instead of Google
- No more sources or evergreens, just posts

Stack
- React Router-based frontend, to be run on Cloudflare worker
- Pocketbase for auth, DB, and S3, running on Google Cloud Run

Both new techs for me :) I'm really enjoying them so far. Pocketbase simplifies so many things. And React Router is much more intuitive than the mess that NextJS has turned into ATP (or I'm just old and not well adapted to App Router).
